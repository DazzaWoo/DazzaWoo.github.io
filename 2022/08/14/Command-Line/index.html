什麼是終端機

一般來說我們在作業系統裡操作都為GUI(圖形使用者介面)，對於新使用者而言，降低使用者的操作負擔之外，圖形介面對於使用者來說在視覺上更易於接受，學習成本大幅下降。而另一種操作模式是使用CLI(命令列介面)，a.k.a CUI(文字使用者介面)，透過此介面輸入指令，電腦接受後再執行指令。
來源維基百科

終端機常用指令

在終端機介面操作經常使用的指令介紹：

MacOS/Linux 指令 代表意思
pwd 目前所在位置
ls 當前資料夾裡所有的檔案
cd [路徑] 切換目錄 [路徑]
mkdir 新增資料夾
touch 新增檔案
cp 複製檔案
mv 移動檔案
rm 刪除
clear 清除畫面上內容

pwd

當前所在位置為：Users/uncledrew

ls

列出當前目錄下的所有檔案及目錄

ls -al: a 顯示有數點開頭的檔案(如：.gitignore)， l 則是顯示檔案權限、擁有者、建立及修改時間

cd

移動到 documents 目錄

mkdir

建立 tmp 目錄

touch

要先移動到 tmp 目錄，再建立 index.html 檔案

cp

從 index.html 複製一份檔名為 01.html

mv

先在 tmp 目錄中建立01、02目錄

分別在01及02目錄中建立檔案

將02.html移動到01目錄

rm

刪除 index.html

rm -r tmp: 刪除 tmp 資料夾，刪除目錄指令需要多 + -r

clear

清除畫面內容
